Dunamis Favicon Generation Instructions

This document provides instructions for creating favicon files from the Dunamis logo for the website.

Required Favicon Files

The following files should be created from the Dunamis logo and placed in the public directory:

  • favicon.ico - Multi-size ICO file (16x16, 32x32, 48x48)
  • icon.png - 192x192 PNG file
  • apple-icon.png - 180x180 PNG file optimized for Apple devices

Method 1: Using Online Favicon Generators

The easiest way to create these files is to use an online favicon generator:

  1. Go to a favicon generator website like RealFaviconGenerator or Favicon.io
  2. Upload the Dunamis logo image (the orange "D" logo)
  3. Configure the settings (maintain the orange gradient colors)
  4. Download the generated package
  5. Extract and place the files in the public directory of the project
Note: Make sure the logo is properly cropped with appropriate padding before generating favicons to ensure it looks good in small sizes.

Method 2: Manual Creation

If you prefer to create the files manually:

  1. Use an image editor like Photoshop, GIMP, or Figma
  2. Create versions of the logo at different sizes (16x16, 32x32, 48x48, 180x180, 192x192)
  3. Save the appropriate sizes in the required formats
  4. Use a tool like ICO Converter to create the .ico file if needed

Testing Favicons

After placing the files in the public directory, you should test them by:

  1. Running the development server
  2. Opening the site in different browsers
  3. Checking how the favicon appears in tabs, bookmarks, and on mobile devices
